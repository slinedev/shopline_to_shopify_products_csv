**使用方法**：
1. 登录 Shopline 后台 https://admin.shoplineapp.com
2. 打开浏览器开发者工具（F12）→ Console 标签
3. 将以上全部代码复制粘贴到控制台，回车执行
4. 脚本会自动分页获取所有商品，生成符合 Shopify 导入模板的 CSV 并下载

**说明**：
- 当前脚本完美支持无变体的简单商品（你的店铺示例就是这种情况）
- 如果未来有变体商品，会在控制台警告并跳过（可后续扩展）
- Vendor、Product category 等字段留空，便于你批量填写或后续调整
- 重量假设 API 返回值为 kg → 转换为 grams（如需调整可修改 `Math.round(p.weight * 1000)`）
- 图片支持多张（会自动生成额外图片行）

如需增加变体支持或其他自定义字段，随时告诉我！
